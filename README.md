# spring-cloud-hystrix-workshop
This repository demonstrates hystrix , turbine and hystrix dashboard

* Turbine monitor stream endpoint:  http://IP-ADDR:9505/turbine.stream?cluster=HYSTRIX-CLIENT-SERVICE
